## Task

Pick ONE common food ingredient or additive and write an interesting, tweet-worthy post about it.

## Requirements

- Choose a different ingredient each time (see recent topics in Context above to avoid repeats)
- Include 2-3 surprising or little-known facts
- Cite specific sources (studies, FDA documents, industry reports, with year)
- Keep it factual and engaging -- not alarmist
- The ingredient should be something most people encounter regularly in packaged foods

## Output Format

Write a single post formatted for Twitter/X:
- Start with a hook line that grabs attention
- Present the facts clearly with line breaks between points
- End with relevant hashtags
- Keep total length under 1500 characters
- Do NOT use thread format -- write it as a single post

Return ONLY the post content. No preamble, no explanation, no meta-commentary.
